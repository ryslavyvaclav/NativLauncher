### NativLauncher
 Is a GUI [Electron](https://www.electronjs.org/) [website](https://ryslavyvaclav.github.io/NativLauncher-web/) wrapper made up using [Nativefier](https://github.com/nativefier/nativefier/) scripts.
 
## Installation
1. Unzip
2. Launch
3. Enjoy
